PM2-ADD-SYNTH

[GENERIC-FUNCTION]

ARGUMENTS:
  - partiels partials [default = nil]
&key  
  - attack partials attack time (s) [default = 0.01]
  - release partials release time (s) [default = 0.01]
  - sr sample rate [default = 44100]
  - res resolution [default = 16]
  - out output pathname [default = "pm2-out.aiff"]
Description:

Realizes additive synthesis using pm2.

The input is a simple list of partials provided as an SDIFFILE or SDIF-BUFFER with 1MRK/1TRC frames.

CHORD-SEQ can also be connected and are iternally converted to an SDIF file as well.
